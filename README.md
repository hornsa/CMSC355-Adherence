# Medication Adherence System

#Overview
This project is a medication adherence system designed to help users manage their prescriptions, track medication usage and improve consistency through reminders and logging.

The system allows users to add medications, record whether doses were taken or missed and monitor adherence over time.

#Features
- Add and remove medications
- Schedule medication times
- Log doses as taken or missed
- Track adherence history
# How to Run
Run the application: python main.py
